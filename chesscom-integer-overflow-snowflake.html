What is Integer Overflow?

When a number exceeds the maximum value its data type can store, the bits wrap around. In a signed integer, the sign bit flips — turning a huge positive into a huge negative.

The Fix: 32-bit vs 64-bit

32-bit Signed Integer

The Real Solution: Twitter's Snowflake IDs

64-bit fixes overflow. But in distributed systems, you also need IDs that are globally unique, time-sortable, and generated without coordination. Snowflake does it all in one 64-bit integer.

Key Takeaways
🚫

Never Use 32-bit for IDs

2 billion feels far until it isn't. Default to 64-bit. Always.

🗹

IDs Are Infrastructure

Your ID scheme affects sharding, sorting, indexing, and coordination.

❄

Snowflake for Scale

Uniqueness + ordering + compactness in 64 bits. Battle-tested.
